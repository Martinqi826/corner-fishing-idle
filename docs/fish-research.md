# 鱼系统设计调研摘要（2026-06-13）

> 4 路并行联网调研（游戏数值 / 中国淡水鱼谱系 / 背包扩容 / 自动化付费门控），本文为已采纳结论的精简版。

## 已采纳的设计决策

### 品阶（6 档，见色识阶）
普通(灰) 58% → 优良(绿) 25% → 稀有(蓝) 11% → 史诗(紫) 4.5% → 传说(橙) 1.3% → 神话(红) 0.2%
- 传说+神话 ≈1.5%，与旧版 4 档的传说概率衔接，玩家体感连续。
- 颜色沿用暗黑/魔兽系行业惯例（白绿蓝紫橙 + 顶层红），玩家见色识阶零教学。
- 后续「幸运」付费/升级可抄 Fisch 公式：有效概率 = Base / (1 + Luck%/100)。

### 鱼谱（28 种，中国北方冬季冰钓真实谱系）
- 普通：白条/麦穗/泥鳅/鲫鱼/鲢鳙/黄颡（冰钓基本盘，鲫鱼是北方冰钓头号目标鱼）
- 优良：雅罗/鲤/草/鳊/青（常见经济鱼）
- 稀有：鲈/三角鲂(法罗)/花䱻(吉花)/翘嘴(大白鱼)/鳜(鳌花) ——黑龙江"三花五罗"名贵层
- 史诗：黑鱼/虹鳟/白斑狗鱼/梭鲈/江团/细鳞 ——冷水掠食层（狗鱼是冰钓主要对象）
- 传说：锦鲤(彩蛋保留)/大马哈鱼/施氏鲟/哲罗鲑 ——洄游名贵冷水鲑科
- 神话：中华鲟/达氏鳇（"活化石"国宝层；野生为一级保护，游戏设定为养殖放流个体）
- 体重区间取真实数据（白条 15g ～ 达氏鳇可达 1 吨）。

### 体重定价（Fisch 模型）
- 卖价与体重线性挂钩：v = lerp(vmin, vmax, 体重在区间内的位置) × 鱼竿增值 × ±8% 抖动。
- 体重 roll 用 k²（偏向小个体）。同种鱼前 ~13% 大个体标「大·」、前 ~2.5% 标「巨物·」。
- 相邻品阶价值倍率 ×3.5~4.5（Stardew 全表 ×167、动森 ×11.5、Fisch ×15~20 的折中），全谱跨度 ≈×5000。

### 背包（Melvor / Stardew / OSRS 综合）
- 起始 20 格（= Melvor 起始值；办公间隙节奏匹配）。
- 整档 +5 格、共 7 次扩容到 55 格；费用 1-2-5 阶梯：100/250/600/1500/4000/10000/25000。
- Melvor 哲学：首扩几分钟产出可买（制造"扩容真值钱"初体验），尾段是长线金币回收池。
- 满包 → 暂停钓鱼（不丢鱼不弹窗，HUD 变橙提示）。Melvor 满仓丢稀有物被社区骂了多年，不学。

### 自动卖鱼 = 高级功能（用户决策，调研佐证）
- 设置面板可见 + 锁图标 + 「敬请期待」——锁定态 UI 最佳实践是"可见+明示"，绝不隐藏。
- 参考门控模型：Egg Inc Pro Permit（一次性买断、免费档折损/付费档全额）、Melvor Auto Eat（天价游戏币三档）。
- 警示（Tiny Pasture/Rusty's/Chillquarium 三个买断桌面挂机均不卖 QoL）：若本体做买断制，自动化建议改为游戏内天价金币解锁，付费只卖内容/外观；若免费发布，自动卖鱼可作 $1.99~2.99 一次性「渔夫执照」。

## 留作后续的调研结论（未实现）

- **星级品质层**（WEBFISHING 模板，Godot 源码可移植）：4 档星 ×1/×1.8/×4/×8，概率由鱼饵决定（逐级 roll 数组），与自动卖鱼一起作付费层卖点。注意别三层倍率相乘（品阶×体重×星级）导致通胀。
- **卖鱼时机决策**：动森 CJ 模式（随机出现的 ×1.5 收购商，挂机党回屏钩子）/ 俄钓4 咖啡馆订单（指定鱼 ×2.5）。
- **收藏锁**（Chillquarium）：给想留的鱼上锁，"一键卖出"跳过锁定鱼——免费底线功能。
- **行为激励扩容**（OSRS PIN +20 格思路）：「连续挂机 7 天 +5 格」「首钓传说 +5 格」。
- **自动化部件作稀有渔获**（Melvor 拾取项链 4.55% 掉率思路）：把自动卖鱼装置的零件做成传说级掉落，与钓鱼主题自洽。
- **巨物纪录轴**：每种鱼的最大体重纪录进图鉴（俄钓4 奖杯线）。
