Corner Fishing Idle

贴在桌面右下角的透明挂机钓鱼小场景。它不是传统全屏游戏，而是一个低打扰桌面伴侣：自动钓鱼、收集图鉴、升级鱼竿与鱼饵，在冬日河岸插画里用轻量动效和柔和音频制造持续陪伴感。

项目总览

106正式鱼种，每种 3 档稀有变体 → 106×3 = 318 格变体收集轴；6 品阶、8 类生态标签。
3可玩钓点（河湾 / 静水湖泊 / 海岸码头），按累计渔获解锁。
7随机事件，数据驱动、低干扰、按钓点触发。
29成就目标，覆盖捕获、财富、图鉴、品相、装备、陈列、变体与猫税。

产品一句话

一个在桌面角落持续运转的冬日钓鱼摆件，玩家不需要高频操作，只需要偶尔打开鱼篓、升级装备、查看图鉴和领取成长反馈。

当前性质

运行时是纯 2D：`Node2D` 主场景、`CanvasLayer` HUD、PNG 图层、Canvas 绘制和 2D shader。美术目标是手绘微写实，不是三渲二管线。

核心玩法

抛竿

进入等待状态，播放轻柔抛竿声和浮漂入水声。

等待

鱼竿等级越高，等待时间越短；鱼篓满时自动暂停。

咬钩

浮漂下沉、涟漪触发，播放 bite 提示音。

上鱼

按鱼竿、鱼饵和权重抽取鱼种、重量、星级品相。

结算

进入鱼篓、更新图鉴纪录、触发成就、可卖出换金币。

鱼篓

展示最近捕获、库存容量、单条出售、批量出售和锁定保留。收鱼郎出现时，卖鱼收益提升至 1.5 倍。

图鉴

按稀有度展示 106 种鱼。已发现鱼记录捕获次数、最大体重、首捕日期，并解锁长期徽章：集齐 10 条、巨物、完美★★★。稀有变体（斑斓/鎏金/七彩，6%/1.2%/0.2%，价值 ×2/5/12）按种记录，106×3=318 格收集轴。未发现鱼保持隐藏剪影；缺图标回退按品阶通用鱼图标。

每日订单

每日 1 单，4 种类型：指定鱼种 / 指定品阶（≥某档）/ 大物（≥某重量）/ 完美品质★★★。交付未上锁的符合渔获获 2.5 倍奖励，进度常驻 HUD，跨日刷新。

鱼篓整理

排序（最新 / 价值 / 品阶 / 重量）与「订单鱼」筛选；收藏锁保留心爱渔获不被批量卖出或订单交付。

活水族箱 🐠

「鱼缸」页签——把心爱渔获放进缸里（最多 8 条），缸内鱼沿正弦路径游动、按方向翻转，点鱼弹出纪录卡（最大体重 / 累计 / 首捕日 / 变体），可捞回鱼篓；鎏金 / 七彩变体带脉动光晕。每条 +观赏卖价加成、封顶 +5%。最契合桌面摆件的健康非数值长线 + 收集深度（取代早期「陈列架」）。

桌面宠物 🐱

渔夫旁的小馋猫，上鱼时偶尔扒拉；2% 概率叼走鱼篓里最廉价的一条杂鱼（卖价 ≤30，珍藏 / 订单 / 锁定绝不动）当趣味事件，计入「猫税」统计与成就。冷调缩小、退作环境配角，不压主体。

专注奖励

窗口失焦且无操作时累计连续「专注」，满 25 / 50 分钟下一竿强制升级渔获（保底极品★★，再保底鎏金变体），切回或操作即清零，每日封顶 4 次——离开越久回来越有惊喜。与手动「专注 / 安静模式」开关是两回事。

昼夜时段

按真实时钟流转 黎明 / 白昼 / 黄昏 / 夜晚——随你这一天自然变化。金色时段（晨昏）咬钩更勤、渔获略增值；夜晚静谧但偶有大物。场景随时段轻染色、HUD 显示，低干扰、零存档。

多钓点生态

从“单钓点挂机”升级为“多钓点生态挂机”。每个钓点用生态标签（river/lake/stream/coast/deep/cold/night/protected）圈定鱼池，带专属随机事件、解锁条件与常驻手感系数；各钓点池约 43~48 种鱼（按 tags∩habitat_tags 求交）。骨架数据化（spot_data.gd / event_data.gd），新增钓点只需加数据 + 一张背景图。

新手河湾 · river_bend

默认解锁，最大鱼池——河 / 湖系杂鱼与经济鱼悉数在此出没。事件：鱼汛 / 顺水鱼群 / 晨雾 / 漂流木箱。

静水湖泊 · still_lake

累计钓到 80 条解锁。偏鲈、鳜、黑鱼、狗鱼、大口黑鲈、鳡、鲟与欧鲶。事件：晨雾 / 寒潮 / 顺水鱼群。

海岸码头 · coast_pier

累计钓到 300 条解锁。偏海鲈、真鲷、黑鲷、带鱼、石斑、马鲛、黄鱼、金枪、旗鱼。事件：涨潮 / 漂流木箱 / 保护鱼放流。

经济与成长

成长系统

系统当前规则

品相与个体

每条鱼会抽重量、价值和星级品相。星级倍率为 1.0、1.8、4.0、8.0；同种鱼中大个体会标记“大”，顶级个体会标记“巨物”。这些规则给挂机收益制造长期波动，也让图鉴纪录有追求空间。

美术资源

美术方向

技术结构

窗口与交互

1040×720 透明画布，主美术偏移到右下角。GL Compatibility 渲染器 + 逐像素透明（无灰底），边缘羽化 shader 融进壁纸。空白透明区鼠标穿透到桌面，仅可见场景 / 按钮参与交互；按住场景空白可拖动，位置存档。

场景渲染

`main.tscn` 使用 `Node2D` + `ScenePainter` + `CanvasLayer`。`scene_painter.gd` 负责背景、雾雪、水光、浮漂、涟漪、灯光、偶发 wildlife、渔夫情绪与桌面宠物猫绘制；活水族箱由 `aquarium.gd` 独立自管动画合成。

数据

`fish_data.gd`（106 鱼 / 生态标签 / 变体 / 按钓点鱼池抽取）、`spot_data.gd`（钓点）、`event_data.gd`（随机事件）、`weather.gd`（昼夜时段）、`aquarium.gd`（活水族箱）、`decor.gd`（陈列收藏）、`orders.gd`（每日订单）、`achievements.gd`（成就）均为 `class_name` 模块，逻辑与数据解耦，便于并行扩展。

存档（v11）

路径 `user://corner_fishing_save.json`，原子写 + .bak 回退。保存金币、装备、库存、图鉴、累计、订单、窗口位置；v8 多钓点、v9 陈列架、v10 稀有变体（var/vmask + best_var）、v11 陪伴向（图鉴首捕日期 fd + 专注奖励 focus_min/rt/rd/pend）。旧档逐版兼容迁移，缺字段无损归零，默认落点 river_bend。

验证工具

`tools/validate_game.gd` 用于无头数据和经济验证；`tools/dev_screenshot.gd` 用于引擎内截图和透明窗口检查。

缺资源回退（不阻塞、不崩）

  • 鱼专属图标缺图 → 回退按品阶通用鱼图标（程序化小鱼剪影）。
  • 宠物猫 sprite 缺图 → 回退程序化猫；渔夫情绪缺帧 → 回退 idle + 变换驱动。

  • 旧存档无新字段（多钓点 / 变体 / 陪伴向）→ 无损归零迁移，默认 river_bend。
