Corner Fishing Idle

一个贴在桌面右下角的透明挂机钓鱼小场景。画面以冬日雪山、松林、河面和安静钓鱼人为核心，左上方向使用雾化透明边缘融入桌面。当前主视觉以正式精修插画品质为标准，不以占位图或程序绘制图作为最终交付。

Godot 透明窗口右下角构图手绘微写实低干扰动效

美术方向

资源遵循 docs/art_direction.md 和 docs/art_asset_audit.md：柔和冬日色彩、主体集中在右下角、边缘透明羽化、UI 像小型桌面组件。动画以水波、浮漂、灯光和轻微收竿为主。
