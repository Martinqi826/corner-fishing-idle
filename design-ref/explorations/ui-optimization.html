卖杂鱼 扩容 全部卖出 · +885
19 标签 → 5 主标签 + 「更多」。背包 / 图鉴 / 订单 / 钓点 为高频入口常驻一行不换行；成就 / 统计 / 鱼缸 / 装备 / 设置收进「更多」。活动态用细金线下划，比整块填色更轻、更耐看。
2概览条取代拥挤的操作行。左侧容量进度条一眼看出满没满；右侧用衬线大字直接给出「可卖总额」——卖鱼决策的关键数字，不再藏在按钮 tooltip 里。
3双列密排 → 单列富信息行。给每条鱼喘息空间：左侧品阶描边图框+ 色条复述稀有度（颜色之外多一重编码，弱色觉也可读），重量/品质/大物标签分层清晰。
4售价 = 金币药丸。价格配金币图标做成主操作键，价值即操作；收藏改为挂锁图标（点亮金色），比「锁」字更直观，也不与品质 ★ 抢符号。高价值/稀有行染极淡品阶色 + 图标辉光，让好鱼自己跳出来。
5底部常驻操作栏。主操作「全部卖出」带上即时收益预览（+885），永远停在拇指可达处，列表再长也不丢。

A金币账条。把最重要的金币 + 鱼篓状态收进一枚半透磨砂胶囊：金币用衬线大字 + 图标，鱼篓配一条容量微条——满篓风险一眼可见。比裸文字更清晰，仍透出壁纸、不堵场景。
B状态拆成轻胶囊。钓点·时段、订单各成一枚低调 chip；订单 chip 带状态点（金=可交付）。信息分层，而非挤成一坨同号文字。
C克制不变。仍是右下角三枚水彩圆钮 + 边缘羽化融入桌面；优化只加了恰到好处的玻璃质感，没有引入新色或抢眼动效——符合「安静陪伴」的定位。
普通 优良 稀有 史诗 传说 神话 鎏金变体 ⚠ 优化稿中的「挂锁」「更多 ▾」为新增 UI 线性图标——属提案新元素，待确认后再纳入图标规范。
