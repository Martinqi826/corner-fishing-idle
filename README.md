# 角落垂钓 🎣

贴在桌面角落、透明融进壁纸、自动挂机钓鱼的水彩风桌面挂件。Godot 4.6 · 2D。

> 灵感：办公时角落里一方静水，垂钓人替你慢慢上鱼。不打扰的陪伴式放置游戏（参照 Rusty's Retirement 生态位）。

## 形态

- 无边框 + 置顶 + **逐像素透明**窗口（GL Compatibility 渲染器，无灰底）。
- **可拖动**：在场景空白处按住左键拖到任意位置；默认右下角，位置存档。
- 空白透明区**鼠标穿透**到桌面；弹出面板在更大的透明窗里自由居中展开。
- 边缘羽化 shader，融进任意壁纸。

## 玩法

- **自动挂机钓鱼**：抛竿 → 等待 → 浮标咬钩下沉 + 涟漪 → 自动上鱼**入鱼篓** → 手动卖出变现，循环往复。
- **28 种鱼 / 6 档品阶**（普通→神话），权重抽取；**体重**影响价值，标「大·/巨物·」体型。
- **星级品质**：无 / 上品★ / 极品★★ / 完美★★★，价值 ×1/×1.8/×4/×8，由**鱼饵**决定概率。
- **鱼篓背包**：起始 20 格、阶梯扩容至 55 格；满篓暂停上钩；**排序（最新/价值/品阶/重量）+ 订单鱼筛选**；收藏锁（锁定不被卖出/交付）。
- **成长**：鱼竿（缩短等待 + 提升高阶概率 + 鱼价加成，成本 200×2^级）；鱼饵（蚯蚓→红虫→活虾→秘制饵，提升星级概率）。
- **每日订单**：每天 1 单，4 类（指定鱼种 / 指定品阶 / 大物≥Wkg / 完美品质），交付未锁渔获按鱼价 ×2.5；HUD 常驻进度，跨日刷新。
- **流动鱼贩**：随机限时收购时段，卖价 ×1.5（HUD 金色提示）。
- **图鉴**：每种鱼记录累计数 + 最大体重 + 长期徽章（集齐 10 条 / 巨物 / 完美）。
- **成就**：16 项（渔获/财富/收集/品阶/品相/装备线），达成发金币奖励。
- **离线收益**：回屏弹「离线小结」面板（时长 / 渔获数 / 最值钱一条 / 合计可卖 / 珍稀清单）；上限 8 小时、50% 效率、受背包格数截断。
- **音频**：UI / 抛竿 / 入水 / 咬钩 / 捕获 / 金币 / 升级 音效 + 循环水声环境音；设置可调静音 / 主音量 / 音效 / 环境音。

## 运行

```sh
# 用 Godot 4.6 打开本目录，或命令行：
godot --path .

# 新克隆首次跑工具前先建缓存：
godot --headless --import
```

## 结构

| 文件 | 职责 |
|---|---|
| `main.gd` | 主控：透明窗形态/拖动、钓鱼状态机、经济、背包/订单/成就/图鉴、面板、存档/离线 |
| `scene_painter.gd` | 场景绘制：Codex 主图 + 动态层（水光/雾/雪/涟漪/灯光/小动物），缺资源回退程序化 |
| `fish_data.gd` | `class_name FishData`：鱼种、品阶、权重、体重定价、星级、鱼饵 |
| `achievements.gd` | `class_name AchievementData`：成就配置 |
| `audio_manager.gd` | Autoload `Audio`：音效/环境音加载与播放、音量/静音、设置持久化 |
| `feather_mask.gdshader` | 屏幕坐标椭圆羽化遮罩，让场景向左上渐隐融桌面 |
| `main.tscn` | 主场景：ScenePainter(材质=羽化) + HUD |
| `assets/art/**` | Codex 美术资源（不在工程逻辑里修改，只加载播放） |
| `tools/` | 无头测试 / 引擎内截图 / 平衡探针 |
| `docs/` | 调研、路线图、美术需求、白皮书 project_overview.html |
