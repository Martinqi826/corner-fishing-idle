# 角落垂钓 🎣

贴在桌面角落、透明融进壁纸、自动挂机钓鱼的水彩风桌面挂件。Godot 4.6 · 2D。

> 灵感：办公时角落里一方静水，垂钓人替你慢慢上鱼。不打扰的陪伴式放置游戏（参照 Rusty's Retirement 生态位）。

## 形态

- 无边框 + 置顶 + **逐像素透明**窗口（GL Compatibility 渲染器，无灰底）。
- **可拖动**：在场景空白处按住左键拖到任意位置；默认右下角，位置存档。
- 空白透明区**鼠标穿透**到桌面；弹出面板在更大的透明窗里自由居中展开。
- 边缘羽化 shader，融进任意壁纸。

## 玩法

- **自动挂机钓鱼**：抛竿 → 等待 → 浮标咬钩下沉 + 涟漪 → 自动上鱼**入鱼篓** → 手动卖出变现，循环往复。
- **多钓点生态**：3 个钓点（新手河湾 / 静水湖泊 / 海岸码头），各有专属鱼池与随机事件，按累计渔获解锁；切换钓点改变鱼池、订单建议、事件与背景。
- **60 种鱼 / 6 档品阶**（普通→神话），每条带生态标签（河/湖/溪/海/深/冷/夜/保护）决定出没钓点；权重抽取；**体重**影响价值，标「大·/巨物·」体型。缺专属图标时回退按品阶的通用鱼图标，不崩。
- **随机事件**（数据驱动）：鱼汛 / 晨雾 / 寒潮 / 顺水鱼群 / 涨潮 / 漂流木箱 / 保护鱼放流，低干扰，按钓点触发，叠加咬钩/价值/运气修正。
- **星级品质**：无 / 上品★ / 极品★★ / 完美★★★，价值 ×1/×1.8/×4/×8，由**鱼饵**决定概率。
- **鱼篓背包**：起始 20 格、阶梯扩容至 55 格；满篓暂停上钩；**排序（最新/价值/品阶/重量）+ 订单鱼筛选**；收藏锁（锁定不被卖出/交付）。
- **成长**：鱼竿（缩短等待 + 提升高阶概率 + 鱼价加成，成本 200×2^级）；鱼饵（蚯蚓→红虫→活虾→秘制饵，提升星级概率）。
- **每日订单**：每天 1 单，4 类（指定鱼种 / 指定品阶 / 大物≥Wkg / 完美品质），交付未锁渔获按鱼价 ×2.5；HUD 常驻进度，跨日刷新。
- **流动鱼贩**：随机限时收购时段，卖价 ×1.5（HUD 金色提示）。
- **图鉴**：60 种鱼，每种记录累计数 + 最大体重 + 长期徽章（集齐 10 条 / 巨物 / 完美）；钓点页显示各钓点鱼种收集进度。
- **陈列/装饰**：把心爱渔获摆上「陈列架」（5 槽，可取下），离开鱼篓永久展示；每件 +1% 全局卖价、封顶 +5%。最契合桌面摆件的健康非数值长线 + 收集深度。
- **成就**：25 项（渔获/财富/收集/品阶/品相/装备/陈列线），达成发金币奖励。
- **离线收益**：回屏弹「离线小结」面板（时长 / 渔获数 / 最值钱一条 / 合计可卖 / 珍稀清单）；上限 8 小时、50% 效率、受背包格数截断。
- **音频**：UI / 抛竿 / 入水 / 咬钩 / 捕获 / 金币 / 升级 音效 + 循环水声环境音；设置可调静音 / 主音量 / 音效 / 环境音。

## 运行

```sh
# 用 Godot 4.6 打开本目录，或命令行：
godot --path .

# 新克隆首次跑工具前先建缓存：
godot --headless --import
```

## 结构

| 文件 | 职责 |
|---|---|
| `main.gd` | `class_name CornerFishing` 主控：透明窗/拖动、钓鱼状态机、经济、背包/成就/图鉴、HUD、卖鱼/升级、存档/离线；面板/事件/钓点/订单均薄壳委托下列模块 |
| `ui_panels.gd` | `class_name UIPanels`：所有面板/页签/卡片/样式构建（静态 g 模式） |
| `events.gd` | `class_name Events`：随机事件管理器（EventData 驱动） |
| `spots.gd` | `class_name Spots`：钓点鱼池/运气增值/解锁/切换/订单候选并集 |
| `orders.gd` | `class_name Orders`：每日订单/周目标/今日统计 |
| `decor.gd` | `class_name Decor`：陈列/装饰系统（陈列架、卖价加成） |
| `scene_painter.gd` | 场景绘制：Codex 主图 + 动态层（水光/雾/雪/涟漪/灯光/小动物），按钓点切底图，缺资源回退程序化 |
| `fish_data.gd` | `class_name FishData`：60 鱼种、品阶、生态标签、权重、体重定价、星级、鱼饵、鱼钩、按钓点鱼池抽取 |
| `spot_data.gd` | `class_name SpotData`：钓点（鱼池 habitat_tags / 解锁条件 / 事件池 / 系数 / 背景） |
| `event_data.gd` | `class_name EventData`：随机事件（鱼汛/晨雾/寒潮/顺水/涨潮/漂流木箱/保护鱼放流） |
| `achievements.gd` | `class_name AchievementData`：成就配置 |
| `audio_manager.gd` | Autoload `Audio`：音效/环境音加载与播放、音量/静音、设置持久化 |
| `feather_mask.gdshader` | 屏幕坐标椭圆羽化遮罩，让场景向左上渐隐融桌面 |
| `main.tscn` | 主场景：ScenePainter(材质=羽化) + HUD |
| `assets/art/**` | Codex 美术资源（不在工程逻辑里修改，只加载播放） |
| `tools/` | 无头测试 / 引擎内截图 / 平衡探针 |
| `docs/` | 调研、路线图、美术需求、白皮书 project_overview.html |
