# 角落垂钓 🎣

贴在桌面右下角、透明融进壁纸、自动挂机钓鱼的水彩风小挂件。Godot 4.6 · 2D。

> 灵感：办公时角落里一方静水，垂钓人替你慢慢上鱼。不打扰的陪伴式放置游戏（参照 Rusty's Retirement 生态位）。

## 形态

- 无边框 + 置顶 + **逐像素透明**窗口，钉在屏幕右下角（自动扣任务栏高度）。
- 空白透明区**鼠标穿透**到桌面，只有场景与按钮可点。
- 边缘羽化，融进任意壁纸。

## 玩法（MVP）

- **自动挂机钓鱼**：抛竿 → 等待 → 浮标咬钩下沉 + 涟漪 → 自动上鱼 → 金币入账，循环往复。
- **12 种鱼 / 4 档稀有度**（普通·稀有·史诗·传说），按权重抽取，传说有全屏金光。
- **三个角落按钮**：
  - 🐟 鱼篓 —— 图鉴：按稀有度配色的鱼种网格，未发现显示 `？？？`，附终身统计。
  - 🎣 鱼竿 —— 升级：缩短等待、提升稀有鱼概率、鱼价加成；费用随等级增长。
  - ⚙ 设置 —— 不透明度滑条、回到右下角、退出。
- **存档 + 离线收益**：每 10s 自动存档 + 退出存档；回来按挂机时长结算渔获（上限 8 小时，50% 效率）。

## 运行

```sh
# 用 Godot 4.6 打开本目录，或命令行：
godot --path .

# 新克隆首次跑工具前先建缓存：
godot --headless --import
```

## 开发工具

```sh
# 无头回归测试（数据 / 抽取分布 / 钓鱼经济 / 鱼竿 / 存档离线）
godot --headless -s tools/validate_game.gd

# 引擎内视口截图（带 alpha，验证透明；输出 docs/img/）
godot --path . -s tools/dev_screenshot.gd
```

## 结构

| 文件 | 职责 |
|---|---|
| `main.gd` | 主控：透明窗形态、钓鱼状态机、经济、反馈、面板、存档/离线 |
| `scene_painter.gd` | 程序化水彩场景：雪山/松林/渐变水面/波光/垂钓人/浮标/涟漪 |
| `fish_data.gd` | `class_name FishData`：鱼种、稀有度、权重抽取、鱼竿数值 |
| `main.tscn` | 主场景：ScenePainter + HUD(金币/toast/按钮/面板) |
| `tools/` | 无头测试 + 引擎内截图 |

> 美术为引擎内程序化（`draw_polygon` 逐顶点 alpha + 多层叠涂）；`assets/` 预留成品 PNG 替换位。
